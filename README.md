# -你的强来了
服务外包
功能
一、建立知识库
1.	构建多岗位面试题库（用于提出问题） 网上找题库，要包含技术知识，项目经历深挖，场景题等 （java后端、web前端…）不同岗位要有差异性，信息化类挑选两个岗位
2.	建立岗位知识库（评估学生的回答）
岗位核心要求、核心技术栈、优秀回答范例、评分维度与规则
二、多模态交互界面
1.语音、文字两种交互形式
2.多轮回答能力，面试官会追问（响应要快速），并控制面试节奏
交互流程参考：
选择面试岗位->首次提问->学生作答->追问->评估生成结果
三、面试分析
1．内容分析，回答的技术正确性
2.表达分析，语速、语言清晰度、自信度等
生成评估报告，包含总分、各维度得分（什么样的维度更好，要包含上面两个大的维度，在此基础上进行细分）、优缺点、改进建议
四、能力提升与反馈
记录面试历史，提供能力成长曲线、个性化建议（可以web搜索推送的学习资料）
可以选择增加管理端、摄像头（一起来想创新点吧）
呈现形式不限，网页、app都可

关键技术内容：
1.	RAG（可用工具/框架实现）向量数据库存储->检索器->大语言模型（生成器）
2.	语音识别与情感分析（可调用api或开源项目）
语音转化为文本；
语音合成，面试官要用语音和学生对话，同时展现文字（如何优化语言和文字同步）
情感分析：自信度、清晰度、情感状态
3.	大模型与对话管理
对话存储、上下文管理、合适的提示词
如何保存完整的上下文，如何处理对话超限，如何生成动态追问的问题
材料提交企业要求：本地知识库资料（用于RAG向量化的文件，格式不限），不涉及知识产权的条件下提交
程序提交：不需要提交项目源代码，程序上传到阿里云或部署程序＋部署文档
